# Tick Tack Toe in terminal

## How to run:

- Install [leiningen](https://leiningen.org/)
- Build uberjar `lein uberjar`
- Run it and enjoy! `java -jar target/clj-ticktacktoe-tui-1-standalone.jar`

## Libraries used:

- [clansi](https://github.com/ams-clj/clansi)
- [jline](https://github.com/jline/)
